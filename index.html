Классификация текста

import requests

def check_spam(user_token, message, model_name="ruSpam"):
    server = "https://neurospacex-ruspam.hf.space/api/check_spam"
    data = {"message": message, "model_name": model_name}
    headers = {"api-key": user_token}

    response = requests.post(server, json=data, headers=headers)
    return response.json()

user_token = ""
message = "This is a test message."
result = check_spam(user_token, message)
print(result)

Example Server Response


{
  "is_spam": 0,
  "confidence": 8.190393145923736e-07,
  "tokens_used": 18,
  "cost": 0.005625
}

Explanation:

  • is_spam: Indicates whether the message is classified as spam (1 for spam, 0 for not spam).
  • confidence: The confidence level of the model in its prediction (closer to 1 means higher confidence in the classification).
  • tokens_used: The number of tokens (words or subwords) used in processing the message for prediction.
  • cost: The cost of performing this prediction, typically measured in API usage or credits.
